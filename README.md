# flask_cookie-api

API (backend python) para o frontend do flask_cookie-clone

## Setup

- criar virtualenv
- `pip install -r requirements-dev.txt`

## Rodar testes

```
make run_tests
```

OU

```

make db_test_up
pytest
make db_test_down

```

## Rodar API

```
docker-compose up -d
flask db upgrade     # 1a vez apenas
flask run
```
